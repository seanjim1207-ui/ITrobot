## 專案概覽

這是一個範例專案說明文件，請替換成你自己的內容。

### 技術架構
- **前端**：React + TypeScript
- **後端**：Node.js + Express
- **資料庫**：PostgreSQL
- **部署**：Docker + AWS

### 專案結構
```
src/
├── components/    # UI 元件
├── pages/         # 頁面
├── services/      # API 呼叫
├── utils/         # 工具函數
└── styles/        # 樣式
```

### 重要注意事項
1. 環境變數放在 `.env` 檔案中
2. 開發時使用 `npm run dev`
3. 正式環境使用 `npm run build`
