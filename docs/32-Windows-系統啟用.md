**症狀:**  內網/外網 windows 系統重灌後沒啟用，右下角出現未啟用浮水印

**解決方法:**
1.可以到此路徑S:\_IT\Microsoft\WIN10_KMS
2.內網的電腦可以執行:"S:\_IT\Microsoft\WIN10_KMS\win10_kms.bat
3.外網的電腦可以執行:"S:\_IT\Microsoft\WIN10_KMS\win10_kms 外.bat"
4.建議可以用記事本打開看一下bat裡面的內容
5.如果在出差在外需要解也可以搜尋:【Windows kms git】
https://github.com/netnr/kms
