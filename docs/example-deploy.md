## 部署流程

### 開發環境
1. 確認所有測試通過：`npm test`
2. 建置專案：`npm run build`
3. 本地預覽：`npm run preview`

### 正式環境部署
1. 合併 PR 到 `main` 分支
2. CI/CD 會自動觸發部署
3. 確認部署狀態：查看 GitHub Actions

### 回滾步驟
如果部署後發現問題：
1. 到 GitHub Actions 找到上一次成功的部署
2. 點擊 Re-run 重新執行
3. 通知團隊成員

### 注意事項
- 部署前務必確認環境變數已設定
- 大版本更新需先通知使用者


### C槽滿了要刪什麼

**症狀:**  同事會問C槽滿了，但不知道刪什麼

**解決方法:**
1.可以先到此路徑:"S:\檢測軟體\硬碟空間\SpaceSniffer"
2.滑鼠右鍵"已系統管理員身分執行"
3.開資料夾找到相對路徑手動刪除
4.通常.cache命名可以刪除

**常見問題:**
1.可以到 C:\Users\使用者名稱
2.點選【檢視】→隱藏的項目→會顯示隱藏資料夾"AppData"
3.點選後點開→local→Temp ，裡面的檔案都可以清空
4.某些同事下班電腦會被抓去當算圖機，如果有moonreder帳戶的temp很多也可以刪除

### F槽滿了請幫我清空
**症狀:**  同事在F槽內刪除掉檔案後，硬碟空間並沒有變多

**解決方法:**
1.原因是F槽的檔案刪除後，會到NAS的垃圾桶，必須清空NAS的垃圾桶
2.下載軟體"putty"，用SSH方式我們將連到主機
3.在【Host Name (or IP address)】格子內輸入: "userdata"
3.出現遠端小黑窗時，帳號:"root" 密碼:"at-ha11@storage"
4.```資源回收桶位置:/mnt/backup/.recycle/```
5.先開啟mis.moonshine.tw查詢同事帳號，再輸入"ls"確認同事資料夾
![[Pasted image 20260409102820.png]]
6.```刪除指令: rm -rf /mnt/backup/.recycle/同事帳號```

### 遠端重開關電腦方式

**症狀**:  同事當機喊機器人重開無效，在你下班或是不在公司，可以透過遠端的方式開關電腦，4樓的電腦基本上都有無線遙控插座，且電腦都有設定通電開機。

**解決方法:**
1.下載【eWeLink/易微聯】，申請帳號，請主管將你的帳號給管理員權限
2.左上角的群組 找到 【My Home】，上面的數字跟座位表的數字相同
3.請對應座位表，找到需要重開的同事座位數字，點下關 後3秒 在點一次打開

### 如何入AD
**症狀:**  新的電腦/重灌後的電腦要入公司AD

**解決方法:**

1.將電腦連上公司內部的網路線
2.對著左下角微軟LOGO右鍵→【系統】→【進階系統設定】→【變更】→確認電腦名稱
→"成員隸屬"裡的【網域】→輸入moonshine.ad
3.輸入你自己有權限的帳號密碼
4.到Active Directory 管理介面，左側欄位有Computers
5.新加入的AD電腦都會到這邊
6.打開左側欄位列表"computer_user"→"Work"群組→"DisableUSB"
7.將新加入的電腦丟到這個群組內

**注意事項**
1.每個資料夾規則都不同，如果是筆電可以丟到"computer_user"→"Surface"
2.如果是會議室電腦，丟到"computer_user"→"meeting"
3.上網機丟到"computer_user"→"VDI"→"VGPU"
4.有時候電腦重灌但原本的電腦名稱已經在AD內，這時要先到AD的這些群組中找到電腦刪除，再重新加入AD

### 如何退AD

**症狀:**  偶爾會遇到某些電腦要退出公司AD

**解決方法:**

1.將電腦連上公司內部的網路線
對著左下角微軟LOGO右鍵→【系統】→【進階系統設定】→【變更】→確認電腦名稱→下面"成員隸屬"勾選【工作群組】→隨意輸入名稱 ex:work
2.輸入你自己有權限的帳號密碼
3.完成

### EPSON系列印表機連線方式

**症狀:**  同事會請你安裝印表機，看同事位置，安裝最近的一台

**解決方法:**
1.EPSOP系列最簡單解法是到windows 設定中找到【印表機】
2.按下新增印表機→點"我想要的印表機未列出"→勾選"依名稱選取共用的印表機
→輸入印表機的IP位置(ex: \\\192.168.1.184\EPSON L5190 Series)→下一步按到底即可完成
3.關於EPSON印表機的IP位置為:
	EPSON L5190: \\\192.168.1.184(USB連接)
    EPSON L6270 : \\\192.168.7.16(USB連接)
	EPSON L15190: \\\192.168.7.25
4.此S槽路徑也有Driver提供安裝:"S:\_IT\Driver\EPSON"
5.關於掃描方式，安裝完驅動後電腦會另外安裝【EPSON SCAN2】程式
6.點開後就可以掃描
7.內網的主機安裝即可，安裝完成後，上網機重新登入後會自動抓取印表機

### HP  M181fw 動畫部印表機安裝 

**症狀:**  動畫部的PM會請你安裝這台印表機 (PS:此台是從IP LAB場域借回不是公司買的)

**解決方法:**
1.安裝印表機驅動 ，到這個路徑：S:\_IT\Driver\HP  
2.HP型號印表機需安裝兩個程式，分別為印表機驅動與ＨＰ掃描專用的程式
3.到路徑後點開資料夾→upd-pcl6-x64-7.9.0.26347  
找到資料夾裡面名稱叫install的檔案，並打開它
4.安裝模式選擇傳統模式
5.點選下一步後，會跳出新增印表機視窗，勾選第四個選項「使用IP位址或主機名稱新增印表機
6.會跳出輸入印表機位址  
	a.裝置類型改成→「TCP/IP裝置」  
	b.主機名稱或IP位址->輸入192.168.1.84  
	c.勾選查詢印表機並自動選取要使用的驅動程式  
	d.點選下一步
7.會跳出安裝印表機驅動程式的視窗 ，印表機會列出HP機器的型號
8.跳出輸入印表機名稱，預設為HP Universal Printing PCL 6
9.新增印表機視窗跳出後，點選「共用這個印表機，讓您網路上的其他人可以找到並使用它」
10.跳出完成視窗即可關閉，
11.安裝掃描驅動
12.到此路徑安裝驅動：\QSAN\Software_IT\Driver\HP  →「Full_Webpack-63.6.6364-SJ0001_Full_Webpack」檔案
13.一路確定後，跳出連線選項點選「手動設定」
14.下一步，點選「有線網路-使用Ethernet纜線將裝置連接到網路」
15.網路檢查的地方會出現機型名稱即可點選下一步完成。
16.總結 若文字太多想看圖片 可以直接提供這篇Knowhow，我有附上圖片。

### Blender (RDP遠端滑鼠偏移問題）


**症狀:**  公司教室電腦都是小電腦遠端機房主機，實習生上blender時遇到滑鼠狀況。

**解決方法:**
1.原因:遠端時，Blender抓不到螢幕的邊界，導致滑鼠相對位置失靈，數值經過滑鼠左移右移都會增加的問題。
2.打開Edit→Preferences→跳出Blender Preferences→ Input→ 在滑鼠欄位找到 Continuos Grab 把打勾取消掉

### windows筆電遠端設定教學

**症狀:**  同事會帶筆電要設定遠端，方便連線回來作業

**解決方法:**
步驟一 
1.下載手機APP imOTP(ios)或是DroidOTP(android)
2.點選Initialize Secret後"搖晃"手機，會產生一組金鑰，記得按右上角的done → Save存檔
3.請想一組4位數的pin碼(ex:1324)，將剛剛的金鑰跟四位數一起給Joe哥設定
4.登入後輸入4位數pin碼，會得到一組動態密碼(30秒更新一次) ，為登入vpn使用

步驟二
1.安裝VPN連線軟體SmartVPN
2.請到[https://ppt.cc/fWsgwx](https://ppt.cc/fWsgwx) 下載(密碼為:moonshine)
3.到雲端硬碟裡 下載"SmartVPN & moonshinem_通用.cfg"
4.安裝完後開啟SmartVPN→ 選取系統設定
5.點瀏覽→ 匯入剛剛下載好的設定檔"SmartVPN & moonshinem_通用.cfg"
6.點選連線,輸入使用者名稱:公司員工帳號，密碼:(手機生成的動態密碼)
7.連上後 狀態會寫:已連接，即完成

步驟三
1.請同事先查看自己的 電腦名稱，右鍵左下角微軟LOGO→ 系統→ 就可以看到裝置名稱
2左下角搜尋列找【遠端桌面連線】→點開後，在中間電腦輸入: 電腦名稱.moonshine.ad
3.點選下方的顯示選項→進階→設定→勾選使用這些RD閘道伺服器設定
→輸入:rdsgateway.moonshine.ad→確定→按下連線
4.會跳出RD閘道伺服器認證，請輸入帳號:moonshine\員工帳號 密碼:員工密碼
5.第一次設定或登入會跳出無法識別，這時請按下檢視憑證
6.視窗跳出後→詳細資料→複製到檔案→下一步→匯出檔案格式→DER編碼二位元X.509(.CER)
7.任選一個電腦資料夾位置，命名檔案後存檔
8.找到剛剛匯出的資料夾位置，執行剛剛的.cer檔→在一般中選擇安裝憑證
9.選擇"本機電腦"→下一步→選擇"將所有憑證放入以下存放區"→按"瀏覽"→選擇"受信任的根憑證授權單位→按下完成。
10.回到遠端桌面連線→按下連線→輸入帳號:moonshine\自己公司帳號 密碼:自己公司密碼→共會需要輸入兩次→完成。

11.需要圖文詳細教學:https://knowhow.moonshine.tw/books/%E5%B1%85%E5%AE%B6%E5%B7%A5%E4%BD%9C
