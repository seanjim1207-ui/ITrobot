**症狀:**  每個專案槽內會有多種專案，沒有參與權限的人會無法打開資料夾

**解決方法:**
1.先請同事找該專案的PM，請PM給專案的權限。
2.開啟左下角的windows搜尋列，找到【 遠端桌面連線】
3.在【電腦】的格子中輸入: "tiger"
4.登入你的帳號後，桌面會有Active Directory 管理介面名為【ProjectGroups】
5.點進去後，在左側欄位找到該專案資料夾，ex: 202507_5Plus5
6.點中間欄位的安全性群組，找到【 成員】，新增同事的帳號即可按下確定。
7.請同事登出登入，再點資料夾看是否成功。
