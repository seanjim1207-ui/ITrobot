## 帳號與權限資訊

### 重要帳號
| 服務 | 說明 | 負責人 |
|------|------|--------|
| AWS Console | 雲端服務管理 | 請聯繫主管取得 |
| GitHub Org | 程式碼倉庫 | 請向 IT 申請加入 |
| Slack | 團隊溝通 | 自行註冊 |

### 權限申請流程
1. 填寫權限申請表（Google Form）
2. 主管審核
3. IT 部門開通
4. 申請人確認

### 注意事項
- 密碼請使用密碼管理器保存
- 離職時所有帳號會自動停用
- 不要將帳號密碼寫在程式碼中
